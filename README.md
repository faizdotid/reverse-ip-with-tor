## Reverse IP
<pre>
Reverse-IP with tor
Working On Linux
</pre>

## HOW TO INSTALL IT
```bash
$ git clone https://github.com/faizdotid/reverse-ip-with-tor
$ cd revTOR
$ apt install tor
$ pip3 install requests[socks]
$ python3 rev.py
```
